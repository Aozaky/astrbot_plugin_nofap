## 指令

- **/nofap mark** : 戒色打卡
- **/nofap update**(or **to**) [**一个数字**] : 直接修改戒色天数
- **/nofap rank** : 查看群内戒色榜
- **/nofap fail**(or **day0**) : 戒色失败，重置天数
